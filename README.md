# Teen Mental Health Dashboard

A Power BI dashboard exploring how social media habits, sleep, stress, and anxiety relate to teen mental health.

## Tools Used
- Power BI
- Excel
- DAX

## Project Overview
This project analyzes teen behavioral and mental health indicators using an interactive dashboard built in Power BI. The dashboard focuses on social media usage, sleep patterns, emotional wellbeing, and depression-related differences.

## Key Insights
- Teens labeled as depressed show higher average social media usage
- Sleep hours are lower among the depressed group
- Stress and anxiety levels are noticeably higher in depressed teens
- Interaction level shows variation in depression rate
